# CNCVerse CloudStream Extension (Clean & Ad-Free Fork)

### Requirements

Cloud Stream ( To install : [View Docs](https://recloudstream.github.io/csdocs/) )

### To Install Clean Repo (Ad-Free & Popup-Free)

1) **Manual Install** (Copy and Paste in **Add Repository** inside CloudStream):

   ```
   https://raw.githubusercontent.com/Subrata96411/CNCVerse-Cloud-Stream-Extension/refs/heads/builds/CNC.json
   ```

2) **Direct Install Link**: [Click to Install in CloudStream](https://cloudstream.cf/add-repo?url=https%3A%2F%2Fraw.githubusercontent.com%2FSubrata96411%2FCNCVerse-Cloud-Stream-Extension%2Frefs%2Fheads%2Fbuilds%2FCNC.json)

### About this Fork
This is a clean, ad-free, and popup-free fork of the original [NivinCNC/CNCVerse-Cloud-Stream-Extension](https://github.com/NivinCNC/CNCVerse-Cloud-Stream-Extension). 

- 🚫 **No Ad Redirection**: Completely removed the 3rd-party website redirection (like `omg10.com`) from every provider's `loadLinks` method.
- 🚫 **No Popups**: Removed the subscription popups and community telegram dialog prompts.
- 🔄 **Automated Daily Sync**: Configured a daily GitHub Actions workflow that automatically pulls upstream changes from the original repository every day at 3:00 AM UTC, strips the ad/popup code patterns, compiles the clean extensions, and updates this repository automatically.
